## Cara Menjalankan

Clone Project

```bash
  git clone https://github.com/zakizakiran/Tugas-Besar-IF5-10123206.git
```

Masuk ke dalam directory

```bash
  cd Tugas-Besar-IF5-10123206
```

Run streamlit

```bash
  streamlit run main.py
```
